# Contribuir a VozVE

Agradecemos tu interés en contribuir a VozVE. Este documento proporciona las pautas para contribuir.

## Cómo Contribuir

1. **Fork** el repositorio
2. Crea una **rama de feature** (`git checkout -b feature/AmazingFeature`)
3. **Commit** tus cambios (`git commit -m 'Add some AmazingFeature'`)
4. **Push** a la rama (`git push origin feature/AmazingFeature`)
5. Abre un **Pull Request**

## Estándares de Código

- Usa **TypeScript** en nuevo código
- Sigue **ESLint** y **Prettier** config
- Escribe **tests** para nuevas features
- Documenta el código públicamente

## Reportar Bugs

Usa GitHub Issues con el template:
- Descripción clara del bug
- Pasos para reproducir
- Comportamiento esperado vs actual
- Screenshots/videos si aplica

## Solicitar Features

Abre una issue con:
- Descripción de la feature
- Caso de uso
- Beneficios

¡Gracias por contribuir! 🙌
